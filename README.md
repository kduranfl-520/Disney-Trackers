# Disney Trackers

A personal offline collection tracker for **Disney Magic Kingdoms** and **Disney Dreamlight Valley**.

This is a single self-contained web app. The entire app lives in `index.html` — no build step, no dependencies, works offline once loaded.

## How it's hosted

This repo is published with **GitHub Pages**. The live app is served from `index.html` at the repository's Pages URL (e.g. `https://USERNAME.github.io/REPO-NAME/`).

## Updating the app

To update to a new version, replace `index.html` with the new file (Add file → Upload files → commit). The home-screen app will pick up the change automatically.

## Backing up your data

Your tracked progress is stored in your browser/home-screen app, not in this repo. Use the **Export backup** button in the app to save a `disney-trackers-backup.json` file, and **Import backup** to restore it.
